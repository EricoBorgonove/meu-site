# Meu Site  

Este repositório contém os arquivos do site [borgonove.tech](https://borgonove.tech), que serve como um espaço para compartilhar conteúdos, informações e projetos.  

## Sobre o Site  

O site é uma página informativa que apresenta conteúdos organizados de forma acessível para visitantes. Ele conta com seções principais, incluindo:  

- **Página Inicial (`index.html`)**: Apresentação geral do site e seus conteúdos.  
- **Página de Contato (`faleConosco.html`)**: Um formulário para que os visitantes possam entrar em contato.  
- **Página de Erro 404 (`404.html`)**: Página personalizada exibida quando um link inválido é acessado.  

Além disso, há um diretório **assets/** onde são armazenados arquivos de mídia e outros recursos essenciais para a exibição do site.  

## Como Acessar  

Para visualizar o site, basta acessar o endereço:  
➡ [borgonove.tech](https://borgonove.tech)  

## Tecnologias Utilizadas  

O site foi desenvolvido utilizando tecnologias web padrão, como:  
- **HTML**: Estrutura das páginas.  
- **CSS**: Estilização para um design responsivo e moderno.  
- **JavaScript (se aplicável)**: Funcionalidades interativas.  
- **Bootstrap**: Framework CSS utilizado para facilitar a criação do layout e tornar o site responsivo.  

## Contato  

Caso tenha alguma dúvida ou sugestão, entre em contato através da [página de contato](https://borgonove.tech/faleConosco.html).
